# img-gallery
Simple PHP image gallery with MongoDB usage.

# Functionalities
- Add jpg/png file
- Put watermark onto file
- Generate miniature of file
- Make private subgallery
- Add image to favourities
- Search for photo with specified name
- User accounts

# Technologies
- PHP 7
- MongoDB
- AJAX
